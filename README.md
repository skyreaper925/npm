# Кинетические солверы уравнения Больцмана (ВКР)

Репозиторий выпускной квалификационной работы (бакалавриат) **«Разработка программных солверов на
суперкомпьютерных системах для решения кинетического уравнения в задачах переноса»**
(Володин М. В., ЛФИ МФТИ, 2026).

Содержит численные решатели уравнения Больцмана, отлаженные на задачах релаксации и применённые к
задаче переноса (ударная волна перед поршнем), а также сам текст диплома, презентацию защиты и отзыв.

## Метод

- **Уравнение Больцмана**, модель **твёрдых сфер**.
- **Консервативный проекционный метод Черемисина**: отклик интеграла столкновений проецируется на узлы
  скоростной сетки так, что масса, импульс и энергия сохраняются машинно-точно, а логарифмическая
  интерполяция гарантирует положительность ФР и выполнение H-теоремы.
- **8-мерные теоретико-числовые сетки Коробова** для интеграла столкновений
- Для **задачи переноса** — расщепление Странга + монотонная TVD-схема адвекции.
- Расчёты выполнены на суперкомпьютерном кластере ЦКП НИЦ «Курчатовский институт» (очередь SLURM,
  `job-array`); структура метода естественно параллелизуема (независимость по ячейкам и узлам Коробова).

## Задачи

|           | Задача                                        | Кратко                                                                     |
|-----------|-----------------------------------------------|----------------------------------------------------------------------------|
| **task0** | Релаксация «квази-шок» (одногазовая)          | бимодальное н.у., изотропизация `T_xx→T`, H-теорема, сравнение с БГК       |
| **task1** | Релаксация бинарной смеси                     | «холодный (T=1) + горячий (T=2)» → равновесие `T=1.5`, рост энтропии       |
| **task2** | Задача о поршне / ударная волна (**перенос**) | профили `n,u,T`, выход на Ренкина–Гюгонио (`n₁≈3`, `T₁≈3.66`, невязка ~2%) |

Каждая задача организована единообразно: `src/` — солвер (`sem1–3` — ядро метода, плюс задача-специфичные
модули), `data/` — результаты (`*.out/*.npz/*.skip`) и графики (`img/`), `docs/` — постановка и заметки,
`tools/` — `cluster.py` и SLURM-скрипт для запуска на кластере.

## Диплом (`diploma/`)

- `thesis/` — LaTeX-исходник ВКР: `thesis.tex` + `parts/*.tex`, рисунки `images/`, `reference.docx`
  (стиль для Word). Сборка → `out/thesis.pdf` и `out/thesis.docx`.
- `slides/` — презентация защиты `defense.tex` → `out/defense.pdf`.

Итоговые файлы лежат в `out/` (`*.pdf`, `*.docx`), все вспомогательные файлы LaTeX — в `auxil/`
(оба правила — в `.gitignore`). `conf.tex` в корне — отдельный конференц-доклад.

## Требования

- **Python 3.13+** с `numpy` и `matplotlib` — для расчётов и графиков:
  ```bash
  pip install numpy matplotlib
  ```
- **MiKTeX** (`pdflatex`) и **pandoc** — для сборки диплома/презентации/Word.
- Пример запуска расчёта: `python task2/src/piston.py --mode smoke` (затем
  `python task2/src/analyze.py task2/data/smoke.npz`).

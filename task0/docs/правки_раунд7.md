Я добавил в notebook соответствующий расчёт и график (`data/img/m4_evolution.png`). На нём для каждого $u$ показаны:

```latex

Релаксацию удобно сравнивать на моменте 4-го порядка по продольной
скорости:

$$
M_{xxxx}(t) = \sum_\alpha \xi_{\alpha,x}^4 \, f_\alpha \, \Delta V.
$$

Для равновесного Максвелла с температурой $T$ и нулевой средней скоростью

На рис.~\ref{fig:m4} приведено сравнение для трёх значений $u$. Видно:

\item Точное решение БГК (пунктир) монотонно убывает к $3 T^2$.

\item Для $u = 1{,}0$ и $u = 1{,}5$ после $t \approx 5$ численная кривая

Качественно картина согласуется с известным результатом: УБ медленнее

порядка.
```

### #6 — недописана переменная интегрирования в H (Q2)

### #16 — графики для модельного уравнения (Q4, формула 3.34)

Формула 3.34 Пособия: $f(t) = f_M + (f(0) - f_M)\,\exp(-t/t_{rel})$, откуда $\ln \delta_k(t) = -t / t_{rel}$. Аналитическая оценка для жёстких сфер при $T = 1$, $n = 1$: $t_{rel} = 5\sqrt{2}\,T / 16 \approx 0{,}44$, что соответствует декременту $\nu_{th} = 16/(5\sqrt{2}\,T) \approx 2{,}26/T$.

$\delta_k(t) = \|f - f_M\|_k / \|f_0 - f_M\|_k$ — нормированное отклонение от равновесного Максвелла, измеренное в трёх концентрических слоях сферы (формула 3.38):

$$
k = 1:\ 0 \le |\boldsymbol\xi| < \xi_{cut}/3;\quad
k = 2:\ \xi_{cut}/3 \le |\boldsymbol\xi| < 2\xi_{cut}/3;\quad
k = 3:\ 2\xi_{cut}/3 \le |\boldsymbol\xi| < \xi_{cut}.
$$

### #3 — «неточность в индексах при дельтах» (Q1, в δ_(μ_ν − s_ν))

Преподаватель видит неточность в индексе `μ_ν − s_ν`. Я смотрел Пособие (стр. 22, формула 3.26) и код (`sem3.py`) — пары `(λ, μ)` и `(λ+s, μ−s)` корректны (импульс сохраняется в каждой паре: $\lambda + \mu = \alpha + \beta$ и $(\lambda+s)+(\mu-s) = \alpha + \beta$).

Вероятная причина замечания — **неполная индексация** в моих формулах: я писал просто $f_\alpha, f_\beta, f_\lambda, f_\mu, f_{\lambda+s}, f_{\mu-s}$, без явного индекса `_ν`, который указан в Пособии (формула 3.26). Заменить блок про обновление f на полностью индексированную форму:

```latex
\delta I_\gamma(f^n) \cdot \tau \;=\; \Delta_\nu\,\bigl(\delta_{\gamma,\,\alpha_\nu} + \delta_{\gamma,\,\beta_\nu}
- (1-r_\nu)\,\delta_{\gamma,\,\lambda_\nu} - (1-r_\nu)\,\delta_{\gamma,\,\mu_\nu}
- r_\nu\,\delta_{\gamma,\,\lambda_\nu + s_\nu} - r_\nu\,\delta_{\gamma,\,\mu_\nu - s_\nu}\bigr),
```

эквивалентно — изменения шести узлов:

```latex
f_{\alpha_\nu},\; f_{\beta_\nu} \;\mathrel{+}=\; \Delta_\nu;\qquad
f_{\lambda_\nu},\; f_{\mu_\nu} \;\mathrel{-}=\; (1 - r_\nu)\,\Delta_\nu;\qquad
f_{\lambda_\nu + s_\nu},\; f_{\mu_\nu - s_\nu} \;\mathrel{-}=\; r_\nu\,\Delta_\nu.
```

То есть на каждой итерации $\nu$ свой набор индексов $(\alpha_\nu,\beta_\nu,\lambda_\nu,\mu_\nu,s_\nu,r_\nu)$. Сохранение импульса: $\lambda_\nu + \mu_\nu = \alpha_\nu + \beta_\nu$ и $(\lambda_\nu+s_\nu) + (\mu_\nu-s_\nu) = \alpha_\nu + \beta_\nu$. Сохранение энергии — через $r_\nu$.

(Если этот вариант не разрешит замечание — попроси точную фразу преподавателя. В коде формула совпадает с Пособием, индексы расставлены.)
